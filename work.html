Overview

An advanced automated system for comprehensive plant health evaluation through leaf analysis, achieving high precision across four key modules.

Key Features

  • Disease Detection (92% accuracy)
    Custom CNN architecture for identifying plant diseases
  • Age Estimation (89% accuracy)
    Web interface for leaf age classification
  • Stress Factor Analysis (91% accuracy)
    Detection of water stress, nutrient deficiency, and pest infestation
  • Remedy Suggestion System
    Targeted solutions with 90% relevance accuracy

Technologies Used

Python, TensorFlow, CNN, Flask, HTML/CSS, Machine Learning

Research Publication

Published in the School of Computer Science and Engineering, VIT Chennai
